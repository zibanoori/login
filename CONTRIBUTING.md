# Contributing to Glassic Login Form

First off, thank you for considering contributing to Glassic Login Form! It's people like you that make the open-source community such a great place to learn, inspire, and create.

## 🛠️ How to Contribute

### 1. Fork and Clone
- Fork this repository by clicking the "Fork" button on the top right.
- Clone your forked repository to your local machine:
  ```bash
  git clone https://github.com/your-username/glassic-login-form.git
  cd glassic-login-form
